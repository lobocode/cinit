## Cinit

This is a tool that automates the creation of a directory structure for projects write in C language. 


## How to install

1. git clone https://github.com/lobocode/cinit
2. cd cinit 
3. make && make install


## How to use

Execute the command below:

```bash
cinit <project-name>
```

Example:

```bash
cinit xablau-project
```
